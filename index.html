Les bases du HTML

Hypertext Markup Language (html)

Le HTML est le langage de programmation lu et interprété par les navigateurs web.
Chaque site internet possède forcément du HTML qui définit sa structure et son contenu.

Chacune de ces pages est ensuite hébergée sur un serveur que l'on peut trouver sur le réseau.
Chaque serveur possède une adresse IP unique, cependant pour des raisons de sécurité, on utilise le DNS ou Domain Name System.
Ce dernier permet de relier une adresse IP à un nom de domaine : l'URL du site.

Par exemple, le DNS permet de faire le lien entre l'adresse IP et le nom de domaine correspondant.

Il est important d'indenter son code.

Validation et contrôle qualité

On peut également utiliser des validateurs de HTML pour vérifier la qualité du code.

En terme de référence, on a le W3C qui fournit des recommandations et des standards pour le développement web.

L'objectif du projet est de créer une page web statique de qualité.

Les wireframes


Avant de commencer une page, il faut définir et penser à l'avance la structure et les blocs du site via des wireframes.
Cela permet notamment d'estimer à l'avance l'espace à attribuer à chaque section et d'ordonner ces dernières pour assurer une cohérence au sein du site.
Un wireframe contient la taille de la police et son type et quelques couleurs mais il doit rester simple.
Il s'agit d'une maquette.

Il existe plusieurs sites et outils pour les crée:

  • Figma
  • pencil.evolus
  • wireframe.cc
  • Balsamiq

Les balises HTML

Une fois l'aspect du site prévu, on commence à structurer son site via des balises HTML.
Ces dernières vont permettre d'organiser aussi bien la succession des parties que l'aspect du texte.

Les titres

Les titres permettent d'organiser les articles et paragraphes : h1, h2, etc. On a 6 niveaux de profondeur pour les titres.

Mise en forme du texte

Il existe plusieurs balises pour mettre en avant le texte :

  • strong (gras)
  • em (italique)
  • u (souligné)
  • s (barré)

Strong et b ont la même fonction mais strong est mieux reconnu par les navigateurs.

D'autre balises permettent aussi d"affiché du code comme code, marqué des citations avec blockquote, marqué des éléments important avec strong ou cité des éléments et sources via cite.

Les listes


Pour faire une liste simple, utilisez la balise ul (pour unordered list) suivi de plusieurs éléments li.
Pour faire une liste ordonnée, utilisez la balise ol (pour ordered list) suivi aussi d'éléments li.

Exemple de liste non ordonnée :

  • Element 1
  • Element 2
  • Element 3

Exemple de liste ordonnée :

  1. Element 1
  2. Element 2
  3. Element 3

Les tableaux

Les balises table, tr, td permettent de créer des tableaux avec des lignes et des colonnes.

  • table : Définit un tableau
  • tr : Définit une ligne d'un tableau
  • th : Cellule d'en-tête de tableau
  • td : Cellule de tableau

On peux également fusionner des via les attribut colspan pour les colonnes et rowspan pour les lignes

Structurer la page

  • section : regroupe une partie du site ayant le même thème
  • article : regroupe une partie indépendante du site, qui peut être extraite et garder son sens
  • nav : les liens de navigation
  • aside : une barre latérale de contenu comme sur Wikipedia par exemple
  • header : à ne pas confondre avec head. Section d'introduction d'un article ou d'une section
  • footer : conclusion d'une section ou article
