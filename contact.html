Les bases du HTML

Le html est le language du programmation lu est interpréter par les navigateur web.
Chaque site interne possède forcement du html qui définie sa structure et son contenue

Chaque une des ces pages est ensuite hébergé sur un serveur que l'on peut trouvé sur la réseaux.
Chaque serveur possède une adresse ip unique, cependant pour des raison de sécurité, on utilise le DNS ou domaine name protocol
Ce dernier permet de relier une adresse IP à un nom de domaine: l'url du site.

Par exemple, le DNS permet de faire le lien entre l'adresse IP et le nom de domaine correspondant.

Il est important d'indenté son code.


On peux également utilisé des validateur de html pour vérifiée la qualité du code.

En terme de référence on à W3C qui fournit des recommandations et des standards pour le développement web.
